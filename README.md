# Art of War

孙子兵法 × AI Agent 组织框架

Apply Sun Tzu's 13 chapters to AI agent organization and orchestration — task assessment, deployment decisions, multi-agent strategy, token cost control, and risk management.

```
claude skill install wanikua/art-of-war
```
